ABCD Library Automation with ISIS

Background information 1
ABCD = Automatisación de Bibliotécas y Centros de Documentación ABCD = a suite of softwares covering all library and documentation centres functions Central : database creation and management, cataloging, import/export, acquisitions, loans, statistics Serials Control OPAC and Site Advanced loans module

The ABCD software suite : scheme

Background information 2
ABCD is FOSS: free and open source a BIREME initiative supported by VLIR/UDC BIREME avails their best existing web-based ISIS applications (SeCS, iAH/Site) but also co-ordinates the programming of the new Central part => All BIREME ISIS-technology culminates into ABCD ! VLIR/UDC (Flemish University Development Co-operation) selected ABCD as preferred ILS and contributes funding and will provide French version plus training materials + pilot workshop through DOCBIBLAS project (E. de Smet)‏

Background information 3
ABCD aims : To provide ISIS-users community with an integrated library system which is suitable also for university libraries as per requirements of VLIR/UDC e.g. MARC21, OAI, Z39.50, thesaurus-support High volumes (at least up to 500K full records)‏ (close to) state-of-the-art functionality Which can be(come) an alternative to KOHA etc. for ISIS-users building on the existing ISIS reality

ABCD Technology 1
WEB-based : The software runs as web-pages in any web-browser (IE, FF, Opera, Chrome, Safari...) served by any web-server (Apache, Xitami, IIS)‏ Client-server can run on 'localhost' (127.0.0.1) on any PC with Windows or UNIX/Linux or on a real server machine Pages are written in PHP (v5.2)‏ and JavaScript E.g. HTML FCKEditor control can be used AJAX provides interactive interface Maximum local data processing by client to minimize Internet traffic Multimedia and hyperlinks are obvious functions

ABCD Technology 2
ABCD is ISIS-based : All data are stored in ISIS-databases Until 2009 : 'classic ISIS-technology' upgraded with CISIS extensions, i.e. MST+XRF with 1Mb records upto 4 Gb databases, IF with 60-character keys From 2009 : 'seamless' migration to NBP with UNICODE ISIS-Script is used for flexible data retrieval (C)ISIS Formatting Language is used throughout Existing ISIS-databases and their structures can be imported

ABCD Technology 3
ABCD uses PHP : Current version 5.2 Free OS software with large developers community and very wide usage Easy installation, often together with Apache, e.g. WAMP, XAMP, EasyPHP... Already offers a wealth of 'controls' (existing codes for almost any popular function), e.g. calendar, HTML Editor etc. Strong links with e.g. (MySQL and) PostgreSQL to be used in e.g. ABCD advanced loans module ABCD uses JavaScript and AJAX as part of its HTML-capabilities for enhanced interactivity and flexibility

ABCD Modules 1 : database management in CENTRAL module
Database creation and management Creation and modification of database structures Can be copied from existing WinISIS-databases or from existing ABCD-databases Can be created 'from scratch' with FDT, FST, FMT and PFT-editors Updating of created database structures FDT, FST, FMT, PFT + record validation Advanced search form (for cataloging) definition Help files for all languages (Spanish, English, Portugese, French, Arabic…)

ABCD Modules 2 : Cataloging in CENTRAL module
Any bibliographic structure can be used Cataloguing with handy toolbar : Browsing MFN's Searching by form, A-Z browser New or copied record, Z39.50 downloading Edit, stop edit, save or delete record Print, tools (import/export, GMOD, locks, IF) Statistics and reports‏ Help pages, Home

ABCD modules 3 : acquisitions with CENTRAL module
Acquisitions management Based on the ‘workflow’ of the process Results in pre-cataloging and creation of loan object

ABCD modules 4 : Loans with CENTRAL module
Fully configurable, incl. catalog databases Functions available : loan, return, reserve, renew, fines/suspensions, overviews

ABCD Modules 5 : Statistics with CENTRAL module
Statistics and reports can be run on any ISIS-database

ABCD Modules 6 : Meta-search OPAC
OPAC searching with iAH

ABCD Modules 7 : Site and Site Manager
The ABCD portal : fully customizable with integrated meta-search OPAC Portal manager : all components can be defined by the interface

ABCD modules 8 : SeCS
Management of and searching journals DB Cardex functions based on publication patterns

ABCD modules 9 : EmpWeb
Advanced loans module introducing extra possibilities for more demanding environments : Multi-server, multi-catalog, multiple policy loans Unlimited no. of rules based on combinations of object- and user-categories Web-services based Works with any DB-driver for web-services (ISIS, MySQL,…) Integration with OPAC and ‘MyLibrary’ function : users can reserve materials and check their status from OPAC
