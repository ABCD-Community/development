Name

File name

Definition data

Filename to which the record containing the document in question is incorporated.

Purpose of field

This field is used to record the file name to enter a document addressed to the analytical level, monograph, collection or serial.

Description of the elements

Multiple occurrences. The file name must appear scrambled. if an information unit has different files, each must have a name that identifies unequivocally. When registering more than one file name, they must register separated by the element that indicates multiple occurrences.
