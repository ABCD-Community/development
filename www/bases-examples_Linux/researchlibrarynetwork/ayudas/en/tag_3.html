Name

Specific location

Definition data

Information on the physical place where the document is analyzed.

Purpose of the field

This field is used to record the location of a treaty at an analytical level, monographic, serial or document collection.

Description of the elements

 Multiple occurrences. Name or code identifying the information unit that physically owns the document analyzed, and corresponding call number. When registering more than one physical location to the same document, the various paragraphs should be shown separated by the element that indicates multiple occurrences

Example

BI: 338.983/C53

D: 19294.01
